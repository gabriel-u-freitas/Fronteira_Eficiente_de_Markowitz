# Fronteira_Eficiente_de_Markowitz
Projeto feito por mim durante o trainee do Clube de Finanças Esag &amp; UFSC para a etapa do núcleo de Riscos &amp; Derivativos em 2021.1, que consiste num código em Python para simular 10.000 alocações de portfólio diferentes para uma carteira de ações e plotar um gráfico de Fronteira Eficiente de Markowitz

A fronteira eficiente é uma importante ferramenta para a **gestão de portfólio**: é um conceito da Teoria Moderna do Portfólio que descreve a curva dos melhores portfólios, oferecendo o maior retorno possível para cada nível de risco ou o menor risco para um dado retorno, através da diversificação inteligente de uma carteira específica de ativos. Ela permite visualizar, em um gráfico de risco (eixo X) vs. retorno (eixo Y), o limite superior das combinações de ativos, indicando carteiras **subótimas (abaixo da fronteira)** e a **seleção ótima (na fronteira)** para investidores com diferentes tolerâncias ao risco, sendo crucial para **otimizar a relação risco-retorno**. 

O código em Python que montei simula 10.000 alocações de portfólio diferentes para uma carteira de ações definidas arbitrariamente pelo usuário e, então, é plotado um gráfico da Fronteira Eficiente de Markowitz, além de retornar os pesos [das respectivas ações] que resultaram no portfólio "mais eficiente", ou seja, de maior shape-ratio
